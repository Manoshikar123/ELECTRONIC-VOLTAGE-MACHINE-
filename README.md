## Project Title

Electronic Voltage Measurement Machine Using Arduino

## Objective 

To measure DC voltage and display the value on an LCD display using an Arduino.

## Components Required

- Arduino Arduino Uno
- 16×2 LCD Display
- 10kΩ Resistor
- Voltage Sensor Module (0–25V)
- Breadboard
- Jumper Wires
- USB Cable
- DC Power Source

## Working Principle

The voltage sensor detects the input voltage.
Arduino reads the analog voltage value.
Arduino converts the value into actual voltage.
The measured voltage is displayed on the LCD screen.

## Circuit Connections

- Voltage Sensor
- VCC → 5V (Arduino)
- GND → GND (Arduino)
- VSS → A0 (Arduino)
- LCD Display
- RS → D12
- EN → D11
- D4 → D5
- D5 → D4
- D6 → D3
- D7 → D2
  
## Output

- Voltage Meter
- V = 12.45 V
  
## Applications

- Battery voltage testing
- Power supply monitoring
- Electronics laboratory experiments
- Industrial voltage measurement
  
## Advantages

- Accurate measurement
- Low cost
- Easy to build
